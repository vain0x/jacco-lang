{command} batch v{version}

複数のコマンドを並列で実行します。

使用例:
    {command} batch

オプション:
    -h, --help          ヘルプを表示します

標準入力:
    1行につき1つの JSON 文字列を受け取ります。

    型:
        {{ id: number, args: string[] }}
    例:
        {{ "id": 1, args: ["build", "main.jacco", "-o", "main.c"] }}

注意:
    コマンドの結果を標準出力に書くと、そのデータがどのコマンドに由来するものか判別することができません。
    そのため、build コマンドの -o 引数のように、結果をファイルに出力するように指定してください。
