{command} batch v{version}

複数のコマンドを並列で実行します。

使用例:
    {command} batch

オプション:
    -h, --help          ヘルプを表示します

標準入力:
    1行につき1つの JSON 文字列を受け取ります。

    型:
        {{ id: number, args: string[] }}
    例:
        {{ "id": 1, "args": ["build", "main.jacco", "-o", "main.c"] }}

注意:
    標準入力に出力されたデータが、どのコマンドに由来するのかを判別することはできません。
    build コマンドの -o 引数のように、結果はファイルに出力するようにしてください。

    標準エラーに出力されたエラーが、どのコマンドに由来するのかも判別できませんが、
    これの対策は未実装です。
