{command} build v{version}

ソースファイルをビルドします。

凡例:
    {command} build <FILE|->

使用例:
    {command} build main.jacco

引数:
    <FILE>              ソースファイル
    -                   標準入力

オプション:
    -h, --help          ヘルプを表示します

標準出力:
    生成されたC言語のコードが標準出力に出力されます。

標準エラー:
    コンパイルエラーやデバッグログなどが標準エラーに出力されます。
